2020-03-31|Nginx

SSL 问题

Nginx 重启时，报错信息：

通常我们会在 server 有如下 Nginx 配置：

访问网页提示错误代码 ERR_SSL_PROTOCOL_ERROR。

原因在于 Nginx 升级到 1.15 版之后，SSL 的配置不再使用 ssl on，把这一句去掉就可以。

这时再执行：nginx -s reload 平滑重启 Nginx 就不会报错了。
